Expense Tracker (CLI)
#Project Description

A command-line based expense management application built using Java. The system allows users to add expenses, view recorded expenses, calculate total spending, and delete entries through a structured menu-driven interface.

#Features

Add new expense with name and amount

View all recorded expenses

Delete an existing expense

Calculate total spending

Menu-driven interaction

Dynamic data storage using ArrayList

Continuous execution until user exits

Simple and user-friendly CLI interface

#Technologies Used

Java

Scanner Class (User Input Handling)

ArrayList

Loops (do-while, for)

Conditional Statements (switch-case, if-else)

Object-Oriented Programming (OOP) Concepts
